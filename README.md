## Developement

```bash
# Clone code
git clone https://github.com/huytbt/3d-room-scence-webpack-react.git

cd 3d-room-scence-webpack-react

# Install packages
npm install

# Start dev server
npm start

# Testing
npm test

# Build
npm run build
```
